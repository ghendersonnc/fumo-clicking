# Fumo Clicking

### Linux
```sh
$ pip -m venv venv
$ source ./venv/Scripts/activate
$ python -m pip install -r requirements.txt
$ python fumo.py
```

### Windows
```sh
> pip -m venv venv
> ./venv/Scripts/activate.bat
> python -m pip install -r requirements.txt
> python fumo.py
```
